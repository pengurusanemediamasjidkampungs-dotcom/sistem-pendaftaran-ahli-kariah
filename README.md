# Sistem Pendaftaran Ahli Kariah (SPAK)

Sistem Pendaftaran Ahli Kariah (SPAK) adalah aplikasi web yang dirancang khusus untuk memudahkan pengurusan dan pendaftaran ahli kariah masjid secara digital.

## Fitur Utama
- **Pendaftaran Ahli:** Borang pendaftaran lengkap untuk ketua keluarga.
- **Pengurusan Keluarga:** Penambahan maklumat pasangan dan anak-anak di bawah satu profil.
- **Dashboard Admin:** Antara muka pengurusan untuk melihat, menapis, dan mengeksport data ahli kariah.
- **Eksport Data:** Memudahkan pihak masjid untuk mendapatkan senarai ahli dalam format CSV/Excel.

## Teknologi yang Digunakan
- **Frontend:** React.js + Tailwind CSS
- **Backend:** Firebase (Store & Authentication)
- **Deployment:** Vercel / Netlify

## Cara Menjalankan Projek
1. `npm install`
2. `npm run dev`

---
Projek ini dibangunkan untuk tujuan pengurusan e-media masjid kampung.
